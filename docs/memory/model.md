# Memory Engine

## TYPES OF MEMORY

### 1. Semantic Memory
- embeddings
- conversation meaning

### 2. Structural Memory
- repo graphs
- ecosystem graphs

### 3. Operational Memory
- logs
- runtime state

## STORAGE

- Qdrant (semantic)
- Neo4j (graph)
- PostgreSQL (metadata)

## RULE

Memory is append-only. No mutation.
